## Recent Releases

<Update label="2024-10-20" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added `antiraid massjoin` command with configurable threshold, kick action, channel lock, and punishment options.
- Introduced VoiceMaster system for automated voice channel management.

## Improvements

- Enhanced embed customization for better visual appeal in moderation logs.
- Optimized leveling system performance for larger servers.

## Bug Fixes

- Resolved issue with command permissions not applying correctly in protected channels.
- Fixed notification delays in real-time engagement tools.

</Update>

<Update label="2024-10-05" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Premium antiraid suite with mass join protection and automated responses.
- Advanced rules command integration for channel-specific moderation.

## Breaking Changes

- Updated command syntax for `rules` and `commands` to use slash commands exclusively.
- Removed legacy prefix support; migrate using `/setup`.

## Bug Fixes

- Corrected embed rendering issues on mobile Discord clients.
- Patched potential exploit in leveling reward distribution.

</Update>

</Update>

## How to Update

Follow these steps to ensure your server runs the latest version of Greed.

<Steps>
  <Step title="Check Current Version" icon="info">
    Use the `/version` command in your server to verify your installed version.
  </Step>
  <Step title="Reinvite Bot" icon="download">
    Visit [https://greed.best/invite](https://greed.best/invite) and re-add Greed with updated permissions.
  </Step>
  <Step title="Run Setup" icon="settings">
````bash
/setup
````
    Re-run the setup command to apply new configurations.
  </Step>
  <Step title="Test Features" icon="check-circle">
    Test new commands like `/antiraid massjoin on --threshold 5`.
  </Step>
</Steps>
